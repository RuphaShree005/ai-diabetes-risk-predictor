# ai-diabetes-risk-predictor
AI-powered Diabetes Risk Prediction System using Machine Learning with interactive UI, risk meter, and explainable insights for better health analysis.

# Features

-  Machine Learning-based prediction (Logistic Regression, Random Forest, KNN)
-  Automatic best model selection using F1-score
-  Color-based risk meter (Low / Moderate / High)
-  Percentage-based risk output
-  Explainable AI insights (why risk is high)
-  Healthy reference ranges for user awareness
-  Clean and interactive UI using Streamlit

# Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy
